# Evolució dels Moviments d'Autodeterminació (SDM)

Aquest repositori conté una anàlisi detallada de l'evolució dels moviments d'autodeterminació (SDM) des de 1945 fins a l'actualitat, basada en el dataset de Sambanis et al.

## Contingut del Repositori

- `SDM_analisi.Rmd`: Script de R Markdown que genera un informe amb 5 figures clau sobre els moviments.
- `SDM.csv`: Base de dades estructurada utilitzada per a l'anàlisi.
- `SDM Coding Notes I.pdf` i `SDM Coding Notes II.pdf`: Notes qualitatives de codificació que justifiquen les dades.

## Figures Incloses

1.  **Evolució Global**: Nombre total de moviments actius per any.
2.  **Dinàmiques per Regió**: Distribució geogràfica dels moviments.
3.  **Violència vs No Violència**: Proporció de moviments que utilitzen tàctiques armades.
4.  **Nous Moviments per Dècada**: Ritme d'aparició de noves demandes.
5.  **Top 15 Països**: Estats amb major presència històrica de moviments.

## Com utilitzar-lo

Per generar l'informe HTML:
1. Obre `SDM_analisi.Rmd` a RStudio.
2. Prem el botó **Knit**.
